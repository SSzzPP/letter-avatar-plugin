## Letter avatar plugin

> If no Gravatar avatar is set, the first character of the nickname will be used as the avatar.

- This plugin is plug-and-play and can be uploaded and enabled directly in the `WordPress` admin panel.

- The plugin's avatar generation code is based on https://github.com/daolavi/LetterAvatar and https://gist.github.com/leecrossley/6027780
